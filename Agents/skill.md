Arquitectura del sistema:
El sistema estará compuesto por las siguientes capas:
1. Capa de presentación: Esta capa se encargará de la interfaz de usuario, permitiendo a los usuarios interactuar con el sistema a través de una aplicación web o de escritorio.
2. Capa de lógica de negocio: Esta capa se encargará de procesar la información y aplicar las reglas de negocio del sistema, como el cálculo de totales, la validación de datos y la gestión de inventario.
3. Capa de datos: Esta capa se encargará de almacenar y gestionar la información del sistema, utilizando una base de datos relacional para almacenar los datos de clientes, productos, ventas y compras.
